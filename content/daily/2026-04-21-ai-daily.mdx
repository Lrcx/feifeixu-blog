# 🚀 大模型与核心产品

# 1. 通义千问Qwen3.6-Max预览版发布

阿里通义千问团队正式发布Qwen3.6-Max-Preview版本，在保持原有架构基础上显著优化推理效率与长文本处理能力，实测在MMLU、GSM8K等基准上平均提升约30%，并原生支持128K上下文窗口与多模态输入增强，适用于复杂任务规划与文档分析场景。

该版本虽为预览版，但已开放部分API调用，标志着通义千问在国产大模型性能追赶国际一线模型（如GPT-4o）上迈出关键一步，对国内开发者构建高精度AI应用具有重要参考价值。

# 2. 阿里发布Wan2.7-Image，图像生成登顶国内盲测

阿里通义实验室正式发布新一代图像生成模型Wan2.7-Image，通过大规模人类偏好对齐，在匿名盲测中综合评分超越国内所有公开模型，成为当前国内首选的图像生成系统。该模型特别强化了对用户意图的细粒度理解，能根据同一提示词生成风格、构图、细节迥异的多样化图像。

这一成果标志着国产AIGC模型在‘个性化生成’能力上实现关键突破，不再局限于单一风格输出，而是真正实现‘千人千面’——不同用户即使输入相同描述，也能获得符合其审美偏好的专属图像，为电商、设计、内容创作等场景带来更高实用价值。

[原文链接](https://www.qbitai.com/2026/04/403733.html)

# 3. 阿里发布Fun-ASR1.5，方言识别超字节腾讯

阿里正式发布语音识别大模型Fun-ASR1.5，作为千问系列端到端ASR的最新版本，该模型在30种语言和7大方言体系上实现突破，尤其在四川话、闽南话等13种方言的识别准确率上全面超越腾讯、字节的同类模型。其字错误率（CER）相较前代降低56.2%，英语和西班牙语识别准确率均超96%，并能自动处理跨语言混说场景，无需预设语言标签。

Fun-ASR1.5采用MoE混合专家架构，结合数十万小时真实语音数据训练，显著提升噪声环境下的鲁棒性，并强化了标点预测与文本归一化能力，适用于会议纪要、法律笔录等高精度场景。目前模型已在魔搭社区开源，开发者可通过阿里云百炼平台调用API，直接服务于中文场景下的语音转写需求。

# 2. 字节海外营收破三成，AI投入致净利暴跌70%
