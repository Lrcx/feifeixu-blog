### 7. Amazon Bedrock AgentCore 发布

Amazon Bedrock AgentCore 发布：Amazon Bedrock AgentCore 是 AWS 推出的编码代理托管平台，为 Claude Code、Codex、Kiro 等 AI 编程代理提供独立的 Linux 微虚拟机环境，支持持久化工作区、交互式 shell、确定性命令执行和安全工具网关，开发者可合上笔记本而不丢失任务状态。

每个会话拥有独立文件系统、网络和凭证隔离，通过 MCP 协议统一接入 GitHub、Jira 等工具，凭证由 AWS Secrets Manager 和 Token Vault 管理，避免敏感信息暴露在 LLM 环境中。

支持挂载 S3 和 EFS 作为共享技能库与缓存，会话可中断后恢复，启动时间数秒内，已集成 CloudWatch 监控与 IAM 身份认证，适用于企业级安全开发场景。
