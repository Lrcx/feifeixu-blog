使用官方稳定长期更新支持断点续传的huggingface-cli工具，莫要使用git clone相关命令

1. 下载工具

```bash
pip install -U "huggingface_hub[cli]"
```

huggingface-cli使用说明：https://huggingface.co/docs/huggingface_hub/guides/cli#huggingface-cli-download

1. 使用huggingface镜像网站

```bash
export HF_ENDPOINT="https://hf-mirror.com"
```

可以配置到系统环境变量中

1. 使用举例

3.1 下载模型 

```bash
## --local-dir 指定本地的存储路径
huggingface-cli download openai-community/gpt2 --local-dir /data/pretrained/gpt2
huggingface-cli download ctheodoris/Geneformer --local-dir ctheodoris/Geneformer

huggingface-cli download meta-llama/Llama-3.2-3B --local-dir meta-llama/Llama-3.2-3B\
huggingface-cli download huggyllama/llama-7b --local-dir huggyllama/llama-7b
huggingface-cli download google-bert/bert-large-uncased --local-dir google-bert/bert-large-uncased
huggingface-cli download FacebookAI/roberta-large --local-dir FacebookAI/roberta-large
huggingface-cli download PKU-Alignment/alpaca-7b-reproduced-llama-2 --local-dir PKU-Alignment/alpaca-7b-reproduced-llama-2
huggingface-cli download Qwen/Qwen3-Embedding-8B --local-dir /data/pretrained/Qwen/Qwen3-Embedding-8B
huggingface-cli download PKU-Alignment/beaver-7b-v1.0-reward --local-dir PKU-Alignment/beaver-7b-v1.0-reward
huggingface-cli download meta-llama/Llama-3.2-3B-Instruct --local-dir meta-llama/Llama-3.2-3B-Instruct
huggingface-cli download Qwen/Qwen2.5-VL-7B-Instruct --local-dir /run/user/0/Qwen/Qwen2.5-VL-7B-Instruct

```

使用huggingface-cli的另一个优点就是会在`.cache/huggingface/` 下维护了一份模型的符号链接，这意味着调用模型就很方便，下面两种方式都可以找到模型

```bash
AutoModelForCausalLM.from_pretrained("/data/pretrained/gpt2")
AutoModelForCausalLM.from_pretrained("gpt2")
```

小米hf token: [HUGGINGFACE_TOKEN_REDACTED]

3.2 下载数据集

```bash
huggingface-cli download InfiniFlow/text2sql --repo-type dataset --local-dir .
```

## 使用aria2下载

```bash
apt-get -y install -qq aria2

# 从huggingface镜像网站下载，huggingface.co一直不通
aria2c --console-log-level=error -c -x 16 -s 16 -k 1M https://hf-mirror.com/ckpt/ControlNet-v1-1/resolve/main/control_v11p_sd15_canny_fp16.safetensors?download=true -d ./models -o control_v11p_sd15_canny_fp16.safetensors
```
